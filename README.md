# Project Overview

This project uses Python to define a student class and then create, manipulate, and store student data for various fictional students. This was good practice for OOP, working with classes, and also to illustrate the concept of how to build logic for interacting with user data.

## Steps for viewing this project

1. Clone or download the repository to a folder in a chosen location.
2. Open the python file in an IDE or text editor of your choosing to review.
3. You may also run the python file in the terminal but you will not see the whole picture of what is happening.
